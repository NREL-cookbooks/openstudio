# Testing
This cookbook uses kitchen to run the tests. To create all instances in the test run

```
bundle install
kitchen converge
```

You can also run this cookbook with vagrant which will install Ruby, EnergyPlus, and OpenStudio.
```
bundle install
vagrant up
```
